# Public release checklist

Before pushing public:

- [ ] Search for personal names, chat IDs, phone numbers, emails.
- [ ] Search for `/Users/`, `~/.env`, API keys, Telegram tokens, bearer tokens.
- [ ] Remove local cron job IDs and private delivery targets.
- [ ] Keep examples generic: `the user`, `origin`, `<your-repo-url>`.
- [ ] Make clear that probe outputs are leads, not verified facts.
- [ ] Make clear the investment/funding section is not financial advice.
